# 1.1.2 智能农业系统中的业务数据采集和处理流程设计

## 题目要求

1. 传感器数据统计：统计每种传感器的数据数量和平均值（截图 1.1.2-1）
2. 按位置统计温度和湿度数据的平均值（截图 1.1.2-2）
3. 数据清洗和异常值处理：标记异常温度（<-10 或 >50）和湿度（<0 或 >100），用前向/后向填充缺失值，保存为 cleaned_sensor_data.csv

## 答案代码

```python
# ====== Cell 1: 读取数据集 ======
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
# 读取数据集 2分
data = pd.read_csv('sensor_data.csv')
```

```python
# ====== Cell 2: 传感器数据统计 ======
# 1. 传感器数据统计
# 对传感器类型进行分组，并计算每个组的数据数量和平均值 3分
sensor_stats = data.groupby('SensorType')['Value'].agg(['count', 'mean'])
# 输出结果
print("传感器数据数量和平均值:")
print(sensor_stats)
```

```python
# ====== Cell 3: 按位置统计温度和湿度数据 ======
# 2. 按位置统计温度和湿度数据
# 筛选出温度和湿度数据，然后按位置和传感器类型分组，计算每个组的平均值 2分
location_stats = data[data['SensorType'].isin(['Temperature', 'Humidity'])].groupby(['Location', 'SensorType'])['Value'].mean().unstack()
# 输出结果
print("每个位置的温度和湿度数据平均值:")
print(location_stats)
```

```python
# ====== Cell 4: 数据清洗和异常值处理 ======
# 3. 数据清洗和异常值处理
# 标记异常值 3分
data['is_abnormal'] = np.where(
    ((data['SensorType'] == 'Temperature') & ((data['Value'] < -10) | (data['Value'] > 50))) |
    ((data['SensorType'] == 'Humidity') & ((data['Value'] < 0) | (data['Value'] > 100))),
    True, False
)
# 输出异常值数量 2分
print("异常值数量:", data['is_abnormal'].sum())
# 填补缺失值
# 使用前向填充和后向填充的方法填补缺失值 4分
data['Value'].fillna(method='ffill', inplace=True)
data['Value'].fillna(method='bfill', inplace=True)
# 保存清洗后的数据
# 删除用于标记异常值的列，并将清洗后的数据保存到新的CSV文件中 4分
cleaned_data = data.drop(columns=['is_abnormal'])
cleaned_data.to_csv('cleaned_sensor_data.csv', index=False)
print("数据清洗完成，已保存为 'cleaned_sensor_data.csv'")
```

## 代码解析

### Cell 1 - 读取数据
- 读取 sensor_data.csv，包含 SensorID、Timestamp、SensorType、Value、Location 共5个字段，10000条记录
- 传感器类型包括：Temperature（温度）、Humidity（湿度）、SoilMoisture（土壤水分）、SoilPH（土壤酸碱度）、Light（光传感器）

### Cell 2 - 传感器数据统计（对应截图 1.1.2-1）
- `data.groupby('SensorType')`：按传感器类型分组
- `['Value'].agg(['count', 'mean'])`：同时计算每组的数据数量和平均值

### Cell 3 - 按位置统计温度和湿度（对应截图 1.1.2-2）
- `data['SensorType'].isin(['Temperature', 'Humidity'])`：筛选温度和湿度类型的数据
- `.groupby(['Location', 'SensorType'])`：按位置和传感器类型双重分组
- `.mean().unstack()`：计算平均值后转为透视表，使温度和湿度成为列

### Cell 4 - 数据清洗（生成 cleaned_sensor_data.csv）
- 标记异常值：温度<-10或>50为异常，湿度<0或>100为异常
- `data['is_abnormal'].sum()`：统计异常值总数
- `fillna(method='ffill')`：前向填充（用前面的值填充缺失值）
- `fillna(method='bfill')`：后向填充（如果前面也没有值，用后面的值填充）
- `data.drop(columns=['is_abnormal'])`：删除临时标记列
- `to_csv('cleaned_sensor_data.csv', index=False)`：保存清洗后的数据
