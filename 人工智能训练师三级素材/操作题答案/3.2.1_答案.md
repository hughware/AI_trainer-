### 完整可运行代码

```python
import onnxruntime as ort
import numpy as np
import scipy.special
from PIL import Image


# 预处理图像
def preprocess_image(image, resize_size=256, crop_size=224, mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]):
    image = image.resize((resize_size, resize_size), Image.BILINEAR)
    w, h = image.size
    left = (w - crop_size) / 2
    top = (h - crop_size) / 2
    image = image.crop((left, top, left + crop_size, top + crop_size))
    image = np.array(image).astype(np.float32)
    image = image / 255.0
    image = (image - mean) / std
    image = np.transpose(image, (2, 0, 1))
    image = image.reshape((1,) + image.shape)
    return image


# 模型加载
session = ort.InferenceSession('onnx/resnet.onnx')

# 加载类别标签
labels_path = 'onnx/labels.txt'
with open(labels_path) as f:
    labels = [line.strip() for line in f.readlines()]

# 获取模型输入和输出的名称
input_name = session.get_inputs()[0].name
output_name = session.get_outputs()[0].name

# 加载图片
image = Image.open('onnx/img_test.jpg').convert('RGB')

# 预处理图片
processed_image = preprocess_image(image)

# 确保输入数据是 float32 类型
processed_image = processed_image.astype(np.float32)

# 进行图片识别
output = session.run([output_name], {input_name: processed_image})[0]

# 应用 softmax 函数获取概率
probabilities = scipy.special.softmax(output, axis=-1)

# 获取最高的5个概率和对应的类别索引
top5_idx = np.argsort(probabilities[0])[-5:][::-1]
top5_prob = probabilities[0][top5_idx]

# 打印结果
print("Top 5 predicted classes:")
for i in range(5):
    print(f"{i+1}: {labels[top5_idx[i]]} - Probability: {top5_prob[i]}")
```

## 代码解析

### 各空白处填写说明

1. **`session = ort.InferenceSession('onnx/resnet.onnx')`**
   - 使用onnxruntime创建推理会话，加载ResNet的ONNX模型文件
   - 注意：考试环境中模型存储在 D:\jupyter_notebook\onnx 目录下，所以路径为 'onnx/resnet.onnx'

2. **`image = Image.open('onnx/img_test.jpg').convert('RGB')`**
   - 使用PIL的Image.open()打开测试图片
   - .convert('RGB') 确保图片为RGB三通道格式（题目代码已有'RGB'参数）

3. **`processed_image = preprocess_image(image)`**
   - 调用已定义的preprocess_image函数进行预处理
   - 预处理包括：resize到256x256、中心裁剪到224x224、归一化、标准化、转置维度

4. **`output = session.run([output_name], {input_name: processed_image})[0]`**
   - session.run()执行模型推理
   - 第一个参数指定输出名称列表，第二个参数是输入字典

5. **`probabilities = scipy.special.softmax(output, axis=-1)`**
   - 使用scipy的softmax函数将模型原始输出转换为概率分布
   - axis=-1 表示沿最后一个维度进行softmax

6. **`top5_idx = np.argsort(probabilities[0])[-5:][::-1]`**
   - np.argsort()返回排序后的索引，取最后5个（最大的5个），然后反转为降序

7. **`top5_prob = probabilities[0][top5_idx]`**
   - 根据top5索引从概率数组中取出对应的概率值

### 图像预处理流程
- Resize到256x256（双线性插值）
- 中心裁剪到224x224（ResNet标准输入尺寸）
- 像素值除以255归一化到[0,1]
- 使用ImageNet均值[0.485, 0.456, 0.406]和标准差[0.229, 0.224, 0.225]进行标准化
- 转置维度从HWC到CHW格式
- 添加batch维度变为(1, 3, 224, 224)

## 交互流程设计（3.2.1.docx内容）

### 图像识别评估系统人机交互最优方案

#### 一、系统概述

本系统基于ResNet深度卷积神经网络模型，通过ONNX格式部署，实现对输入图像的智能分类识别。系统采用图形化用户界面（GUI），提供直观的交互体验。

#### 二、交互界面设计

界面采用左右分栏布局：
- **左侧区域**：图像上传与预览区，支持拖拽上传或点击选择文件
- **右侧区域**：识别结果展示区，以柱状图和列表形式展示Top5分类结果
- **底部区域**：操作按钮区（上传图片、开始识别、清除结果、导出报告）

#### 三、最优交互流程

1. **图像输入**：用户通过"上传图片"按钮或拖拽方式选择待识别图片，系统在左侧区域实时预览图片缩略图
2. **参数设置（可选）**：用户可设置输出Top-K数量（默认为5），调整置信度阈值
3. **一键识别**：点击"开始识别"按钮，系统自动完成图像预处理（resize、裁剪、归一化）和模型推理
4. **结果展示**：右侧区域以柱状图直观展示Top5类别及其概率值，同时以列表形式给出详细信息
5. **结果导出**：支持将识别结果截图保存或导出为CSV/PDF报告

#### 四、优化策略

- **批量处理**：支持多张图片批量上传和识别，提高工作效率
- **历史记录**：保留最近的识别记录，方便用户回溯查看
- **反馈机制**：用户可对识别结果进行"正确/错误"标记，用于后续模型优化
- **实时提示**：在处理过程中显示进度条和状态提示，提升用户体验
