### 完整可运行代码

```python
import onnxruntime as ort
import numpy as np
import scipy.special
from PIL import Image


# 预处理图像
def preprocess_image(image, resize_size=256, crop_size=224, mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]):
    image = image.resize((resize_size, resize_size), Image.BILINEAR)
    w, h = image.size
    left = (w - crop_size) / 2
    top = (h - crop_size) / 2
    image = image.crop((left, top, left + crop_size, top + crop_size))
    image = np.array(image).astype(np.float32)
    image = image / 255.0
    image = (image - mean) / std
    image = np.transpose(image, (2, 0, 1))
    image = image.reshape((1,) + image.shape)
    return image


# 加载模型
session = ort.InferenceSession('onnx/flower-detection.onnx')

# 加载类别标签
with open('onnx/labels.txt') as f:
    labels = [line.strip() for line in f.readlines()]

# 获取模型输入和输出的名称
input_name = session.get_inputs()[0].name
output_name = session.get_outputs()[0].name

# 加载图片
image = Image.open('onnx/flower_test.png').convert('RGB')

# 预处理图片
processed_image = preprocess_image(image)

# 确保输入数据是 float32 类型
processed_image = processed_image.astype(np.float32)

# 进行图片识别
output = session.run([output_name], {input_name: processed_image})[0]

# 应用 softmax 函数获取识别分类后的准确率
accuracy = scipy.special.softmax(output, axis=-1)

# 获取预测的类别索引
predicted_idx = np.argmax(accuracy[0])

# 获取预测的准确值（转换为百分比）
prob_percentage = accuracy[0][predicted_idx] * 100

# 获取预测的类别标签
predicted_label = labels[predicted_idx]

# 输出预测结果，包含百分比形式的概率
print(f"Predicted class: {predicted_label}, Accuracy: {prob_percentage:.2f}%")
```

## 代码解析

### 各空白处填写说明

1. **`session = ort.InferenceSession('onnx/flower-detection.onnx')`**
   - 使用onnxruntime加载花朵检测模型
   - 考试环境中模型路径为 D:\jupyter_notebook\onnx

2. **`with open('onnx/labels.txt') as f:`**
   - 此处空填 `open('onnx/labels.txt')`
   - 打开类别标签文件，读取1000个ImageNet类别名称

3. **`image = Image.open('onnx/flower_test.png').convert('RGB')`**
   - 使用PIL打开花朵测试图片
   - 题目代码已有 .convert('RGB')

4. **`processed_image = preprocess_image(image)`**
   - 调用已定义的预处理函数
   - 预处理流程与ResNet相同：resize到256、中心裁剪224、归一化、标准化

5. **`output = session.run([output_name], {input_name: processed_image})[0]`**
   - 执行模型推理

6. **`accuracy = scipy.special.softmax(output, axis=-1)`**
   - softmax将原始输出转为概率分布

7. **`predicted_idx = np.argmax(accuracy[0])`**
   - 获取概率最大的类别索引

8. **`prob_percentage = accuracy[0][predicted_idx] * 100`**
   - 取出最高概率并乘以100转换为百分比

9. **`predicted_label = labels[predicted_idx]`**
   - 根据索引从标签列表中获取花朵类别名称

### 图像预处理流程
- 与3.2.1 ResNet相同的预处理管线
- Resize到256x256 -> 中心裁剪到224x224 -> 归一化到[0,1] -> ImageNet标准化 -> HWC转CHW -> 添加batch维度
- 最终输入shape为(1, 3, 224, 224)

## 交互流程设计（3.2.4.docx内容）

### 花朵智能识别系统人机交互最优流程

#### 一、系统概述

本系统基于深度卷积神经网络训练的flower-detection模型，通过ONNX格式部署，能够识别多种花朵类别，输出预测类型和识别准确率。适用于植物学研究、园林管理、自然教育等场景。

#### 二、交互界面设计

界面采用卡片式布局：
- **顶部导航栏**：系统名称、功能切换（单张识别/批量识别/拍照识别）
- **主体左侧**：图片输入区，支持拖拽上传、点击选择、摄像头拍照三种方式
- **主体右侧**：识别结果卡片，展示花朵名称、识别准确率（百分比）、花朵百科信息
- **底部区域**：识别历史记录滚动条

#### 三、最优交互流程

1. **图片输入**：
   - 用户通过拖拽或点击上传花朵照片
   - 或直接使用手机/摄像头拍摄花朵，系统实时获取图像
   - 预览区显示上传的花朵图片

2. **智能识别**：
   - 点击"开始识别"按钮（或拍照后自动触发）
   - 系统自动完成图像预处理（resize、中心裁剪、归一化）
   - 模型推理并输出分类结果

3. **结果展示**：
   - 以卡片形式展示：花朵名称 + 识别准确率（百分比形式，如"daisy: 95.32%"）
   - 附带该花朵的简要百科介绍（花期、产地、特征等）
   - 显示Top3可能的花朵类别及其概率

4. **结果应用**：
   - 支持一键保存识别结果到本地
   - 支持分享至社交媒体
   - 支持添加到个人花卉收藏夹

#### 四、优化策略

- **离线识别**：模型本地化部署，支持无网络环境下使用
- **知识图谱联动**：识别结果关联花卉知识库，提供养护建议
- **AR增强**：手机端支持AR模式，在实景中标注花朵名称
- **批量处理**：支持文件夹批量导入图片，自动识别并生成分类报告
