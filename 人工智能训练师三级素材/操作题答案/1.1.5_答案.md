# 1.1.5 智能交通系统的数据采集、处理和审核流程设计

## 题目要求

1. 数据采集：读取 vehicle_traffic_data.csv，显示前5行（截图 1.1.5-1）
2. 数据清洗与预处理：删除缺失值、数据类型转换、删除异常值（年龄18-70、车速0-200、距离1-1000、时间1-1440），保存为 cleaned_vehicle_traffic_data.csv
3. 数据合理性审核：审核年龄/车速/距离/时间的合理性，标记不合理数据（截图 1.1.5-2）
4. 数据统计：每种交通事件次数（截图 1.1.5-3）、不同性别平均车速/距离/时间（截图 1.1.5-4）、不同年龄段驾驶员数（截图 1.1.5-5）

## 答案代码

```python
# ====== Cell 1: 数据采集 ======
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 1. 数据采集
# 从本地文件中读取数据  2分
data = pd.read_csv('vehicle_traffic_data.csv')
print("数据采集完成，已加载到DataFrame中")

# 打印数据的前5条记录 2分
print(data.head())
```

```python
# ====== Cell 2: 数据清洗与预处理 ======
# 2. 数据清洗与预处理
# 处理缺失值（删除）  2分
data = data.dropna()

# 数据类型转换
data['Age'] = data['Age'].astype(int)       #Age数据类型转换为int 1分
data['Speed'] = data['Speed'].astype(float)     #Speed数据类型转换为float 1分
data['TravelDistance'] = data['TravelDistance'].astype(float)     #TravelDistance数据类型转换为float 1分
data['TravelTime'] = data['TravelTime'].astype(float)     #TravelTime数据类型转换为float 1分

# 处理异常值  2分
data = data[(data['Age'].between(18, 70))  &
            (data['Speed'].between(0, 200)) &
            (data['TravelDistance'].between(1, 1000)) &
            (data['TravelTime'].between(1, 1440))]

# 保存清洗后的数据  1分
data.to_csv('cleaned_vehicle_traffic_data.csv', index=False)
print("数据清洗完成，已保存为 'cleaned_vehicle_traffic_data.csv'")
```

```python
# ====== Cell 3: 数据合理性审核和数据统计 ======
# 3. 数据合理性审核
# 审核字段合理性 1分
unreasonable_data = data[~((data['Age'].between(18, 70)) &
                           (data['Speed'].between(0, 200)) &
                           (data['TravelDistance'].between(1, 1000)) &
                           (data['TravelTime'].between(1, 1440)))]
print("不合理的数据:\n", unreasonable_data)

# 4. 数据统计
# 统计每种交通事件的发生次数  2分
traffic_event_counts = data['TrafficEvent'].value_counts()
print("每种交通事件的发生次数:\n", traffic_event_counts)

# 统计不同性别的平均车速、行驶距离和行驶时间  2分
gender_stats = data.groupby('Gender')[['Speed', 'TravelDistance', 'TravelTime']].mean()
print("不同性别的平均车速、行驶距离和行驶时间:\n", gender_stats)

# 统计不同年龄段的驾驶员数  5分
age_bins = [18, 26, 36, 46, 56, 66, np.inf]
age_labels = ['18-25', '26-35', '36-45', '46-55', '56-65', '65+']
data['AgeGroup'] = pd.cut(data['Age'], bins=age_bins, labels=age_labels, right=False)
age_group_counts = data['AgeGroup'].value_counts().sort_index()
print("不同年龄段的驾驶员数:\n", age_group_counts)
```

## 代码解析

### Cell 1 - 数据采集（对应截图 1.1.5-1）
- `pd.read_csv('vehicle_traffic_data.csv')`：读取车辆行驶数据集，包含 VehicleID、DriverName、Age、Gender、Speed、TravelDistance、TravelTime、TrafficEvent 共8个字段，1000条记录
- `data.head()`：显示前5行数据

### Cell 2 - 数据清洗与预处理（生成 cleaned_vehicle_traffic_data.csv）
- `data.dropna()`：删除缺失值行
- `.astype(int)` / `.astype(float)`：数据类型转换
- `.between(min, max)`：筛选合理范围内的数据
  - 年龄：18-70岁
  - 车速：0-200 km/h
  - 行驶距离：1-1000 km
  - 行驶时间：1-1440分钟（24小时）
- `to_csv('cleaned_vehicle_traffic_data.csv', index=False)`：保存清洗后数据

### Cell 3 - 数据审核和统计（对应截图 1.1.5-2 到 1.1.5-5）
- 合理性审核：对清洗后的数据再次审核（清洗后应无不合理数据，输出为空 DataFrame）
- `data['TrafficEvent'].value_counts()`：统计每种交通事件（Normal、Accident、Traffic Jam、Breakdown）的发生次数
- `data.groupby('Gender')[['Speed', 'TravelDistance', 'TravelTime']].mean()`：按性别分组统计平均车速、距离、时间
- `pd.cut(data['Age'], bins=age_bins, labels=age_labels, right=False)`：按年龄区间分组
- `.value_counts().sort_index()`：统计各年龄段人数并排序
